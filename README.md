# E-commerce SQL Analysis: Brazilian Retail Insights

## Project Overview
Analysis of 100K+ e-commerce transactions from 2016-2018 to uncover customer behavior patterns, revenue drivers, and retention trends.

## Business Questions
1. Customer Lifetime Value by Acquisition Channel
2. Monthly Cohort Retention Analysis
3. Product Affinity Analysis
4. RFM Customer Segmentation
5. Seasonal Trends & Forecasting

## Tech Stack
- MySQL 8.0
- Dataset: Olist Brazilian E-commerce (Kaggle)
- 100,000+ orders, 99,000+ customers
